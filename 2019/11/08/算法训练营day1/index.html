五遍刷题法

第一遍

  • 读题5-10分钟，尽可能多想解法，想不出来不纠结，下一步
  • 看解法理解
  • 默写出来

    第二遍

  • 空白文件自己写出解法
  • leetcode上提交跑代码
  • 多尝试几种解法，看看时间空间复杂度

    第三遍

  • 过一天之后重刷

第四遍

  • 过一周后重刷

第五遍

  • 面试前一周恢复训练

核心

  • 重复多次练习
  • 不舒服就对了

切题四件套

  • clarification 多读几遍题目
  • possible solutions 尽可能想多种可能解法
  • coding 编码
  • test case 测试用例
