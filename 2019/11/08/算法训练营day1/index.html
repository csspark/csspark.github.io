数据结构

一维数据结构

基础：数组array、链表linked list
高级：栈stack，队列queue，双端队列deque，集合set，映射map

二维数据结构

基础：树tree、图graph
高级：二叉搜索数，红黑树，AVL，堆heap 等等

特殊数据结构

位运算，布隆过滤器
LRU cache

算法

基础

  • if else
  • for
  • 递归 recursion
    找重复单元

    高级

    搜索：depth first search 深度优先搜索，Breadth first search，etc
    动态规划：Dynamic Programming
    二分查找 binary search
    贪心算法 greedy
    数学math，几何geometry操作

deliberate practicing 刻意练习

五遍刷题法

第一遍

  • 读题5-10分钟，尽可能多想解法，想不出来不纠结，下一步
  • 看解法理解
  • 默写出来

    第二遍

  • 空白文件自己写出解法
  • leetcode上提交跑代码
  • 多尝试几种解法，看看时间空间复杂度

    第三遍

  • 过一天之后重刷

第四遍

  • 过一周后重刷

第五遍

  • 面试前一周恢复训练

核心

  • 重复多次练习
  • 不舒服就对了

切题四件套

  • clarification 多读几遍题目
  • possible solutions 尽可能想多种可能解法
  • coding 编码
  • test case 测试用例
